# 📧 Phishing Email Detection & Awareness System

## 📌 Overview

This project focuses on identifying phishing emails and understanding common attack patterns. Phishing is a type of cyber attack where attackers trick users into sharing sensitive information such as passwords, OTPs, or banking details.

---

## 🎯 Objective

* Detect phishing emails
* Analyze suspicious patterns
* Classify emails as Safe / Suspicious / Phishing
* Provide awareness and prevention tips

---

## 🪜 Methodology

### 🔹 Step 1: Data Collection

Collected sample phishing emails from online resources and spam folders.

### 🔹 Step 2: Email Analysis

Each email was analyzed based on:

* Sender email address
* Suspicious links
* Urgency or threatening language
* Requests for sensitive information
* Grammar and formatting errors

### 🔹 Step 3: Classification

Emails were classified into:

* ✅ Safe
* ⚠️ Suspicious
* 🚨 Phishing

### 🔹 Step 4: Reporting

Prepared a structured report including analysis and prevention tips.

---

## 📊 Sample Analysis

| Email            | Problem Identified | Classification |
| ---------------- | ------------------ | -------------- |
| Bank Alert       | Fake link          | Phishing       |
| Job Offer        | Asking money       | Phishing       |
| OTP Scam         | Asking OTP         | Phishing       |
| Delivery Message | Suspicious link    | Phishing       |
| Password Reset   | Fake link          | Phishing       |

---

## 🛡️ Prevention Tips

* Do not click unknown links
* Always verify sender email address
* Never share OTP or passwords
* Avoid urgent or threatening messages
* Use spam filters
